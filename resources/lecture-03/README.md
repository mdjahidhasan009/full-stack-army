## Components of a Programming Language
- Input(Std I)
- Processing(Showhow)
    - Foundational
        - Variable
        - Statement/Expression
        - Operator
        - Logic
        - Function
        - Array, Key-Value Pair, DS
    - Structural
        - OOP
        - Functional
        - Design Patterns
        - Design Principales
        - DSA(Not dependes on Programming Languages) 
- Output(Std O)
